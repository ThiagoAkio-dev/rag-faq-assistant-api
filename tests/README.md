# RAG FAQ Assistant API

API em Python com FastAPI para ingestão de documentos curtos, indexação vetorial e resposta a perguntas com recuperação de contexto.

## Stack
- Python
- FastAPI
- Qdrant
- SQLite
- pytest

## Funcionalidades
- Upload de documentos
- Chunking de texto
- Indexação vetorial
- Endpoint `/ask`
- Logs simples em SQLite
